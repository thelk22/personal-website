## 🚀 Getting Started
1. Clone the repository:
   ```sh
   git clone https://github.com/thelk22/personal-website.git
   cd personal-website
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Run the development server:
   ```sh
   npm start
   ```
4. Open your browser and visit `http://localhost:3000`
